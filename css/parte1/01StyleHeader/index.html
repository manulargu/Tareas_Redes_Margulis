¿Cómo aplicar estilos en el header?

Para aplicar estilos en el header, tenes dos opciones, la primera, es agregar una etiqueta style dentro del mismo archivo html, si lo queres hacer mejor, tenes que crear un archivo css aparte y enlazarlo en el head de tu documento.

En este caso, se ha creado un archivo styles.css que contiene los estilos necesarios para este ejercicio.

El archivo styles.css se encuentra en la misma carpeta que este archivo HTML.

Esto es mejor ya que podes dar un mismo estilo a todos los parrafos, a todos los h1, h2, o lo que vos quieras.
