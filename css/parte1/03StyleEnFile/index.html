¿Para que aplicar styles en el file?

En este ejercicio, usaremos un link rel para poder poner todos los estilos directamente en un archivo .css, y no tener que hacerlo en el html

Esto nos permite mantener nuestro HTML más limpio y separado de la presentación visual. Puede parecer que no cambia mucho, pero cuando tengamos una pagina mas robusta, quedaría muy desordenado y se acomplejaria darle estilos a cada elemento de forma individual.
